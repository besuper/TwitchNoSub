# TwitchNoSub

Watch any subscriber-only VODs on Twitch, integrated in the website. Supports all Twitch features.

Support Chromium-based browsers and Firefox.

## Download and installation

##### Chromium-based browsers
Download the latest release in the [releases section](https://github.com/besuper/TwitchNoSub/releases) or clone the Git repository.

You have to install the extension manually:

- Open the "Extensions" page
- Make sure **Developer mode** is enabled
- Click **Load unpacked extension** and select the unzipped folder of the extension.

Your Chromium-based browser may be able to pack the extension to get a .crx file that you can drag and drop inside extensions page to install it. It removes the need to have a dedicated directory for the extension.

##### Firefox
Download the latest .xpi file in the [releases section](https://github.com/besuper/TwitchNoSub/releases).

- Drag and drop the .xpi file on Firefox's "Extensions" page
- Click on "Add" in the little confirmation popup
